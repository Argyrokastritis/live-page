Βιογραφικό

Ο Αργυροκαστρίτης Ιωάννης - Σπυρίδων είναι πτυχιούχος του πενταετούς τμήματος Μηχανικών Ηλεκτρονικών Υπολογιστών και Πληροφορικής του Πανεπιστημίου Πατρών (C.E.I.D) με Integrated Master στον τομέα της Μηχανικής των Υπολογιστών. Με εμπειρία στα ERP συστήματα(Navision ERP) και στην ανάπτυξη web εφαρμογών τόσο για το frontend όσο και για το backend. Πρόθυμος να ενταχθεί σε μία αξιόπιστη εταιρεία για να ξεκινήσει μια παραγωγική, δια βίου καριέρα. Εκτεταμένες γνώσεις σε πολλούς τομείς της μηχανικής υπολογιστών, όπως αλγόριθμοι ταξινόμησης, δομές δεδομένων και σχεσιακές και μη σχεσιακές βάσεις δεδομένων (MySQL, MongoDB και Firebase). Έχει υλοποιήσει επίσης διάφορα ερευνητικά έργα, όπως: Ανάπτυξη Full stack web εφαρμογής σε Node.js και MongoDB σχετική με λειτουργικότητα που αφορά την διαδικασία ανάθεσης και διεκπεραίωσης διπλωματικών εργασιών, Ανάπτυξη web πλατφόρμας εύρεσης εργασίας που έχει υλοποιηθεί ως Full Stack εφαρμογή με Backend: Spring Boot και MySQL database και στο Frontend: React με REST API επικοινωνία με το backend. Ανάπτυξη android εφαρμογής με το android studio για μια εφαρμογή σχετική με ανταλλαγές συγγραμμάτων, project πολυδιάστατων δομών δεδομένων για αξιοποίηση αλγορίθμων δομών δεδομένων όπως (k-nearest neighbors,quad trees και άλλα), project αλγορίθμων παράλληλης επεξεργασίας με χρήση νημάτων, υλοποίηση έργων DSP(Digital Signal Processing) με το MATLAB και πολλά άλλα.

Τα έργα του μπορείτε να τα βρείτε σε αυτόν τον σύνδεσμο στο GitHub.

Biography

Graduate of the five-year Department of Computer Engineering and Informatics at the University of Patras (C.E.I.D), holding an Integrated Master's degree in Computer Engineering. Experienced in ERP systems (Navision ERP) and full stack web development (frontend and backend). Eager to join a reputable company to begin a productive, lifelong career. Also possesses extensive knowledge in various areas of computer engineering, including sorting algorithms, data structures, and both relational and non-relational databases (MySQL, MongoDB, Firebase). Furthermore he has developed multiple research projects, such as: Full stack web application in Node.js and MongoDB for managing thesis assignment and completion processes, Job search web platform implemented as a full stack application with Spring Boot backend and MySQL database, and React frontend with REST API integration, Android application using Android Studio for textbook exchange functionality, Multidimensional data structure project utilizing algorithms like k-nearest neighbors and quad trees, Parallel processing algorithms using threads, DSP (Digital Signal Processing) projects using MATLAB.

Portfolio of projects available on GitHub.
